# Social Network API

## Purpose
Use MongoDB to create a social network API to handle large amounts of unstructured data!

## Installation
- clone the repo
- npm init --y
- npm i
- open cmd prompt and type "mongod" (keep this open)
- npm start
- run your commands in insomnia or similar software :)
